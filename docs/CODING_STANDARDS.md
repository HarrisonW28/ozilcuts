# Coding Standards

## Backend
- Laravel 12
- Service-layer architecture
- Policies for authorization
- DTOs where appropriate
- Queued jobs/notifications
- API versioning

## Frontend
- Next.js 15
- Tailwind
- shadcn/ui
- Framer Motion
- TypeScript strict mode
- Responsive-first

## Styling
- Tailwind first
- BEM where custom CSS is required
- No deeply nested selectors
- Consistent spacing/typography
