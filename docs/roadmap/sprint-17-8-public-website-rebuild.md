# Sprint 17.8 — Public Website Rebuild

## Goals
Create a premium editorial-style public website focused on visual trust and booking conversion.

## Product Direction
The website should feel:
- luxury
- minimal
- editorial
- visual-first
- modern

Avoid:
- generic local-business websites
- content-heavy SEO pages
- cluttered layouts

## Core Pages
- homepage
- services
- gallery
- barber profiles
- contact/location
- booking page

## Homepage Sections
- hero section
- featured services
- barber highlights
- visual gallery
- reviews
- booking CTA
- location/hours

## Focus Areas
- typography
- whitespace
- visual storytelling
- haircut photography
- CTA hierarchy
- responsive layouts

## Frontend Tasks
- redesign homepage
- redesign navigation
- redesign gallery layouts
- improve image presentation
- improve mobile responsiveness
- improve booking CTA visibility
- optimize dark mode presentation

## UX Requirements
- mobile-first layouts
- reduce text density
- premium visual hierarchy
- high-quality image presentation

## Constraints
- minimal CMS complexity
- no blog/content bloat
- preserve existing routing architecture

## QA Checklist
- mobile QA
- image responsiveness QA
- dark mode QA
- performance QA
- tablet QA

## Acceptance Criteria
- significantly improved first impression
- stronger booking conversion feel
- premium visual identity established
